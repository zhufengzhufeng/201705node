## 添加到暂存区
```
git add ./ -A / 文件名
```
## 添加到历史区
```
git commit -m '消息'
```
## 查看版本信息
```
git log --grep / --author
```

## git diff
- git diff 比较工作区和暂存区
- git diff --cached 比较暂存区和历史区的不同
- git diff master 比较工作区和历史区的不同

## 用暂存区回滚工作区的内容 
```
git checkout 文件名
```

## 回滚暂存区
```
git reset HEAD 文件名
```

## 回滚版本
```
git reset --hard 版本id
```

## 查看所有日志
```
git reflog
```

## 分支管理
- 默认没有分支，只有提交过才拥有分支
- 只要提交一次就有分支了

## 查看分支
```
git branch 
```

## 创建分支
```
git branch 分支名
```

## 切换分支
```
git checkout 分支名
```
## 删除分支
```
git branch -D dev
```

## 创建并切换
```
git checkout -b 分支名
```
